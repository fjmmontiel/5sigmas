Como aprenden (y como se convierte en sistema)
Supervisado / No supervisado / Auto-supervisado / RL describen de donde sale el "profesor".
Idea
Tienes ejemplos con etiqueta: x -> y. El modelo aprende a predecir y.
Ejemplo
  • Objetivo: detectar spam.
  • Datos: emails con etiqueta spam/no spam.
  • Salida: probabilidad de spam.
  • Feedback: correcciones de usuario -> nuevas etiquetas.
Cuando encaja
Cuando puedes conseguir etiquetas fiables y medir el error.
Idea
Solo tienes x. Buscas estructura: grupos, patrones, anomalias.
Ejemplo
  • Objetivo: segmentar clientes.
  • Datos: comportamiento sin etiquetas.
  • Salida: clusters con perfiles similares.
  • Feedback: negocio valida y ajusta variables o numero de clusters.
Cuando encaja
Cuando no hay etiquetas, pero si quieres organizacion o deteccion de rarezas.
Idea
El propio dato contiene la respuesta. El sistema se fabrica y* a partir de x.
Ejemplo
  • Objetivo: completar frases / modelar contexto.
  • Datos: textos sin etiquetar.
  • y*: siguiente palabra o fragmento real del texto.
  • Feedback: comparacion directa con el token real.
Cuando encaja
Cuando tienes mucho dato sin etiquetar y quieres preentrenar un modelo general.
Idea
No hay "respuesta correcta" por ejemplo. Hay objetivo y recompensa; se aprende probando acciones.
Conceptos
  • Agente, entorno, accion, recompensa, retorno.
Ejemplo
  • Objetivo: mantener un vehiculo en el carril.
  • Acciones: girar, acelerar, frenar.
  • Recompensa: centrado = +1, salirse = -1.
Cuando encaja
Problemas secuenciales: importa el efecto a largo plazo de decisiones.
Idea
IA en la practica = sistema: objetivo medible + datos + salida + metricas + feedback.
Mini-checklist
  • Objetivo: que optimizas (fraude evitado, precision, coste, latencia...).
  • Metricas: como decides si mejora o empeora.
  • Feedback: como llegan correcciones (humanas, logs, chargebacks...).
  • Ciclo: monitorizar -> ajustar/reentrenar -> validar -> desplegar.
Mensaje clave
"Entrene un modelo" != "tengo IA en produccion". Sin operacion y mantenimiento, es una demo.
