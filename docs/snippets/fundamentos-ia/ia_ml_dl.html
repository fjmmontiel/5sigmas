IA
El paraguas: decidir y planificar
Que es
Marco amplio: tecnicas para que un sistema razone, planifique o decida. Puede usar reglas, busqueda, optimizacion o (a veces) aprendizaje.
Entrada tipica
reglas · restricciones · objetivos
Salida tipica
decisiones · planes · acciones
Capacidades
-Aprende de datos -Red neuronal -Genera contenido
  • Aspiradora robot: ruta eficiente
    Planifica por donde pasar para cubrir la casa evitando obstaculos.
  • Sudoku/ajedrez: mejor jugada
    Explora opciones y elige la que maximiza una puntuacion.
  • Turnos en una tienda
    Asigna horarios cumpliendo reglas y restricciones.
  • Reparto: ruta con varias paradas
    Optimiza el orden de visita para minimizar tiempo o kilometros.
  • Soporte por reglas (FAQ)
    Si detecta X, recomienda Y siguiendo un flujo definido.
  • Semaforos coordinados
    Ajusta ciclos con logica de control para reducir atascos.
ML
Aprender patrones desde ejemplos
Que es
Rama de IA donde el sistema aprende de datos para generalizar a casos nuevos, sin programar reglas para cada situacion.
Entrada tipica
datos (etiquetados o no) · variables · metrica
Salida tipica
prediccion · clasificacion · agrupacion
Capacidades
✓Aprende de datos -Red neuronal -Genera contenido
  • Demanda del super
    Predice cuantas unidades se venderan para ajustar stock.
  • Riesgo de impago
    Estima probabilidad de impago para decidir condiciones.
  • Fraude en tarjeta
    Detecta transacciones sospechosas comparando con patrones.
  • Churn (bajas)
    Predice quien podria cancelar una suscripcion.
  • Segmentos de clientes
    Agrupa perfiles similares para ofertas mas relevantes.
  • Recomendacion de productos
    Sugiere articulos segun comportamientos parecidos entre usuarios.
DL
ML con redes neuronales profundas
Que es
Subconjunto de ML basado en redes profundas, especialmente util para imagen, audio y lenguaje (datos complejos).
Entrada tipica
muchos datos · computo · embeddings
Salida tipica
deteccion · reconocimiento · representaciones
Capacidades
✓Aprende de datos ✓Red neuronal -Genera contenido
  • Desbloqueo por cara
    Reconoce un rostro comparandolo con una representacion aprendida.
  • Transcribir notas de voz
    Convierte audio en texto aprendiendo patrones acusticos.
  • Leer texto en fotos (OCR)
    Detecta y reconoce letras en imagenes.
  • Deteccion de objetos
    Identifica y localiza coches, personas, senales.
  • Modo noche en camara
    Reduce ruido y recupera detalle combinando informacion.
  • Fotos similares en galeria
    Encuentra imagenes parecidas comparando embeddings visuales.
GenAI
DL que genera contenido
Que es
Subconjunto (habitual) de DL orientado a crear contenido nuevo: texto, imagen, audio o codigo, condicionado por un objetivo y contexto.
Entrada tipica
prompt · contexto · ejemplos
Salida tipica
texto · imagenes · audio · codigo
Capacidades
✓Aprende de datos ✓Red neuronal ✓Genera contenido
  • Redactar un email
    Genera un borrador con objetivo y tono especificados.
  • Resumir un documento
    Condensa puntos clave en un texto breve y estructurado.
  • Reescribir con otro tono
    Ajusta estilo: mas formal, mas breve o mas persuasivo.
  • Crear una imagen
    Genera una imagen coherente a partir de una descripcion.
  • Asistente de codigo
    Propone funciones, corrige errores y sugiere refactors.
  • Guion para una presentacion
    Produce un esquema y notas del ponente a partir del tema.
